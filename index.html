    Wag - React Application

    Wag is a react application project that creates a hypothetical business which uses geolocation data to allow dog owners to build digital fences and allow walkers to be tracked while they walk the dog within that fence. Owners can watch the walks in near real time and walkers are notified if they leave the digital fence.

    This project uses Firebase for storage and authentication as well as the real time database. ReactJS and React Router as well as several mapping npm libraries
